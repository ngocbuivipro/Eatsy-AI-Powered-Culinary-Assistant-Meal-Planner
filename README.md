# 🍽️ Eatsy – AI-Powered Culinary Assistant & Meal Planner

## 📌 Overview

Eatsy is an AI-powered culinary assistant designed to help users make smarter cooking decisions, reduce food waste, and personalize their daily meal planning experience.

Unlike traditional recipe applications that provide static content, Eatsy leverages Large Language Models (LLMs) and a structured recipe database to deliver context-aware recommendations based on available ingredients, user preferences, and dietary goals.

---

## 🚀 Key Features

* 🥗 **Ingredient-Based Recipe Matching**
  Suggest recipes based on ingredients users already have.

* 🤖 **AI Cooking Assistant**
  Interactive chatbot powered by LLMs to answer cooking questions, suggest substitutions, and generate meal ideas.

* 🎯 **Personalized User Profiles**
  Tailored recommendations based on dietary preferences, health goals, and cooking habits.

* 🔍 **Smart Filtering System**
  Filter recipes by meal type, difficulty, calories, and dietary constraints.

* 📅 **Meal Planning Support**
  Generate optimized meal plans for individuals or families.

### AI Integration

* OpenAI API / Google Gemini (LLM-based assistant)

---

## 🧠 System Architecture

Eatsy follows a modular full-stack architecture:

* **Frontend Layer:** Handles user interaction and UI rendering
* **Backend Layer:** Manages business logic and API endpoints
* **Database Layer:** Stores user profiles, recipes, and preferences
* **AI Layer:** Provides intelligent, context-aware responses
